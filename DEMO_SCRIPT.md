# Vivid - Demo Video Script (2-3 Minutes)

## 🎬 Video Structure (180 Seconds Total)

### 1. Hook - The Problem (0:00-0:20)
**Visual:** Person overwhelmed with sticky notes, screenshots, voice memos, and notes scattered across apps.

**Audio (Voiceover or Text):**
> "We capture thoughts everywhere - sticky notes, voice memos, screenshots, random notes in different apps. But when we need them most, they're impossible to find. What if your notes could think back?"

**Call to Action:** Show Vivid logo with tagline: "Vivid - Your AI-Powered Second Brain"

---

### 2. Core Feature #1: Capture Anything (0:20-0:50)
**Visual:** Quick montage showing:
1. Tap blue "+" button
2. **Text Note:** Type "New React Native performance idea..." → Save → AI automatically tags it "idea" and "React Native"
3. **Voice Memo:** Tap microphone → Record → AI transcribes and categorizes
4. **Photo:** Take photo of whiteboard → AI extracts text and tags
5. **Link:** Paste URL → AI generates preview and summary

**Audio:**
> "With Vivid, capture anything in seconds. Text, voice, photos, PDFs, links - all automatically organized by AI."

---

### 3. Core Feature #2: AI Chat with Knowledge Base (0:50-1:30) **THE KILLER FEATURE**
**Visual:** 
1. Tap "Chat" tab
2. Type: "What did I learn about React Native last week?"
3. AI responds: "Based on your notes from March 15th and 20th, you captured 3 insights about React Native performance..."
4. AI shows citations: "[1] React Native Performance Tips (insight)"
5. Follow-up: Tap "Show me connections" → Graph view appears

**Audio:**
> "Here's the magic. Ask Vivid anything about your knowledge. It searches your entire second brain, finds connections you forgot, and cites specific notes."

**Key Point:** Show the **citations** and **graph connections** - this is the wow moment!

---

### 4. Core Feature #3: Smart Organization (1:30-1:50)
**Visual:**
1. Show "Library" tab with organized sections
2. Tap "Graph" tab → Show visual network of connected ideas
3. Tap a node → Shows the note details
4. Tap "Settings" → Toggle dark mode → App transforms

**Audio:**
> "Vivid automatically organizes everything - by type, tags, even visual connections. Switch to dark mode. It just works."

---

### 5. Advanced Feature: Task Reminders (1:50-2:10)
**Visual:**
1. Capture screen → Toggle "Mark as Task"
2. Set reminder: "Tomorrow 9 AM"
3. Show notification appearing on phone lock screen

**Audio:**
> "Turn any note into a task with smart reminders. Vivid even sends daily AI insights based on your knowledge."

---

### 6. Technical Excellence (2:10-2:30)
**Visual:** Split screen showing:
- Left: App working offline (airplane mode on)
- Right: Sync happens when back online
- Show code snippet: "Offline-first with Expo SQLite + Supabase sync"

**Audio:**
> "Built with cutting-edge tech. Works offline, syncs when online. Your data, always available."

---

### 7. Closing - Call to Action (2:30-3:00)
**Visual:**
1. Montage of all screens: Onboarding → Capture → Chat → Graph → Settings
2. App logo with tagline
3. Show hackathon branding

**Audio:**
> "Vivid. The AI-powered second brain that never forgets. Capture anything, chat naturally, find connections you never knew existed. Download now."

**Text on Screen:**
- "Vivid - AI Second Brain"
- "Built for Expo Track - Hackathon 2026"
- "GitHub: [your-repo-url]"
- "Demo: vivid.app/demo"

## 🎥 Production Notes

### Visual Style:
- **Clean, minimalist** aesthetic (show the beautiful UI)
- **Smooth transitions** between screens (use screen recording)
- **Zoom in** on key features (AI chat, graph view)
- **Dark mode** segment (shows polish)

### Audio:
- **Background music:** Soft, tech-inspired but calm
- **Voiceover:** Clear, warm, confident (not robotic)
- **Sound effects:** Subtle button clicks, swooshes for transitions

### Key Moments to Emphasize:
1. **00:50-1:30** - AI Chat (THE WINNING FEATURE)
   - Show citations in chat
   - Show graph connections
   - This proves "Innovation in AI usage"

2. **00:20-0:50** - Auto-organization
   - Show AI tagging in action
   - Proves "Real-world usefulness"

3. **02:10-2:30** - Offline-first
   - Proves "Technical quality"

### Recording Tips:
- Use **screen recording** on your actual phone (shows it's real)
- **Zoom in** on text/buttons (make it readable)
- **Landscape mode** for better visibility
- **High quality** (4K if possible, 1080p minimum)
